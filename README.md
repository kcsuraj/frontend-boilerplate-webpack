
<h1>Frontend Boilerplate with Webpack</h1>

## Prerequisites

- [nodejs](https://nodejs.org/)

## Usage

Clone the repo:

```bash
git clone
```

Cd into the project folder

```bash
cd frontend-boilerplate
```

Install yarn modules

```bash
yarn
```

Build production

```bash
yarn build
```

Build version is available in `dist` folder.

## Development

### Server

For development you can use:

```bash
yarn start
```
